# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1 as its code of conduct.

The full text is published at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, assume good faith, and focus on the work. Project spaces include this repository's issues, pull requests, discussions, and any operator-side communication tied to a deployment.

## Reporting

Concerns can be reported privately to **ZeroDay@fubar.cyou**. Reports will be reviewed and responded to within a reasonable timeframe. Maintainers may take any action they consider appropriate, up to and including a ban from project spaces.
